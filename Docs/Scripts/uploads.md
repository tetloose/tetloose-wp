# Wordpress Uploads

## Push

`yarn uploads:push`

Follow the prompts:

This will do the following:

1. Zip local uploads
2. Rsync uploads to server
3. Unzip and replace uploads

## Pull

`yarn uploads:pull`

Follow the prompts:

This will do the following:

1. Zip server uploads
2. Rsync uploads to local
3. Unzip and replace uploads
